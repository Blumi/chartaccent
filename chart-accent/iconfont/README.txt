How to generate the icon font:

1. Update the font in Adobe Illustrator by editing modeltracker-icons.pdf.
2. Save the pdf file, then save it as svg with individual canvases, you'll get multiple SVG files, each corresponds to a single icon.
3. Open the https://icomoon.io/app web app.
4. Upload the SVGs to the app.
5. Export the font as zip file by clicking the "Generate Font" button.
6. Download the zip file, extract the css file and fonts/, put them into this folder.
7. Make some changes:
    - Change font family to "modeltracker-icon".
